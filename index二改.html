-----前言-----

此教程是为了帮助一些新手玩家遇到的困难。

-----第一章：进入篇-----

对于某些人来说，加入服务器是个难题。请按你遇到的问题对症下药。

  1. 安装不上

    答：使用群文件中的[安卓14可用]版本。

  2. 服务器IP端口是什么?

    答: IPmc-kkk.top 端口:19132

  3. 显示login timeout

    答：这个的意思是登录时间过长，可以尝试[重启]你的[手机]。

  4. 显示“无法连接至互联网，请检查您的互联网再试”

    答: 你可能使用的是移动数据网络，这玩意不能锁定IP，无法连接。可以尝试一个有网络的场所或者是下载一个流量客户端。如果你已经连接了互联网，可以断网重连。

  5. 显示“无法连接至世界”

    答: 重新启动[手机]可以解决一部分问题。

  6. 显示“名称不能用Steve”

    答: steve是游戏内的初始名称。你需要更改为除了默认名称外的所有名称。更改教程：主界面>选项>名称>点击更改。

  7. 显示“您没有给手机存储权限”

    答: 手机设置>应用管理>找到游戏客户端(如Minecraftpe等)>权限管理>手机存储>打开权限。

-----第二章：注册篇-----

既然你已经进入了服务器内部，那么你必须要注册才能游玩游戏。

  1. 第一步：打开游戏界面上方的[聊天]按钮。（如果使用的是iOS，请点[暂停]旁边的按键[聊天]）
  2. 第二步：查看聊天框内的内容，请查看自己的聊天框内的内容，然后看以下内容。
  3. 显示“请输入你的密码”，但是你没有设置密码

    答：这是因为你使用的名称已经被其他人注册了，服务器内每个名称代表着一个账号。解决方法：退出并重新更改一个名字（更改方法请看上面）。

  4. 显示“正在注册你的账号，请先输入你的名称”

    显示这个就可以看第三步了。

  5. 第三步：按照以下的流程注册账号
    1. 你设置的什么[名字]就在[聊天框]内输入你的[名字]。
    2. 你需要设置一个密码，首先，你需要设置一个密码（例：123456（不建议））。
    3. 你需要再次确认你设置的密码（例：123456（不建议））。

    如果显示“注册完成,欢迎来到生存斧”之类的文字，就说明你已经注册成功了。

-----第三章：登录篇-----

你已经学会注册了，若你想再次进入服务器，那你一定要查看以上的教程。

  1. 输入你设置的[密码]（例：123456（不建议））。
  2. 登录成功，你可以进行游玩了。

登录成功之后您可以通过主城前往新手教程区域了解信息。

-----第四章：常识篇-----

服务器内有很多东西你必须要知道，接下来是服务器内的一些机制。

  1. 重启时间：每90分钟会重启一次。
  2. 清理时间：约每10分钟会清理一次掉落物，清理提醒时间约是10秒前。
  3. 底部状态栏内容：
    • 位置(你所在的区域坐标)
    • 服务器群(加入可以获得最新动态)
    • 人数(人数超过50，服务器将异常卡顿)
    • 网络(查看当前延迟)
    • 金币(你的金币数量)
    • 方块ID(方块的编号)

-----第五章：指令篇-----

你已经可以正常游玩了，不过指令是这服务器的进阶玩法。接下来，这些是基本的指令和用途。（提示: 括号内的注释不用写）

  1. /sethome或/安家(安置你的家,安置一次花费300金币)
  2. /home或/回家(回到你安置的家)
  3. /w 世界名
    • 1.world(生存区)
    • 2.nether(地狱)(只能在前面输入/w 使用)
    • 3.ender(假装是末地)
    • 4.parkour(跑酷世界)(已失效)
    • 5.zc(主城)
    • 6.zy(资源区)
    • 7.pvp(玩家对战)
    • 8.shop(世界商店)
  4. /ws(打开世界玩家交易中心,不过一定要查看价格,小心破产)
  5. /cd(箱子菜单,里面有很多生存用得到的功能。)
  6. /res tp 领地名(传送到指定领地)
  7. zp i(用方块点击要查看的地方的方块状态,如果已经打开查询模式，再次输入该命令可以关闭查询模式)
  8. /tpa 玩家名称(传送到别人那里)
  9. /tpahere 玩家名称 (让别人传送到你这)
  10. /tell 玩家名称 内容(和别人说悄悄话)
  11. /prefix 想要改的称号(输入这串指令，可以让你的称号改成你喜欢的称号)
  12. 其余指令请输入/help 1~11来了解
  13. 告示牌指令如下:
    (第1行:名称可有可无)
    ( 第2行:plcmd )
    ( 第3行:命令 )
    ( 第4行:命令扩展 )

-----第六章：金币篇-----

金币是服务器里面的硬通货，你应该学会如何使用。

  1. 要把钱给别人,输入/支付 玩家名 数量
  2. 要获得免费的钱,输入/qd并点击黄色的羊毛
  3. 要通过劳动赚钱,可以前往shop世界的回收商店
  4. 要使用钱可以前往shop世界的出售商店
  5. 输入/ws可以购买其他玩家的商品,一定要注意价格,否则容易破产
  6. 想把自己的东西卖到玩家商店?手持你要卖的物品,输入/ws sell 价格
  7. 输入/查询 玩家名字 可以查看指定玩家的金币数量
  8. 输入/财富榜 可以查看财富的排行榜
  9. 服务器里面超过十万金币就需要扣税了。
    请务必注意此项！

以上就是所有关于金币的帮助了。祝你早日成为富豪！

-----第七章：世界传送篇-----

你不会还在傻傻的输入/w 世界名 这么麻烦的传送吧？观看以下世界传送帮助可以让你更快的传送世界

以下是可以在聊天框内输入并传送的世界

  1. pvp(玩家对战)
  2. parkour(跑酷世界,可以在聊天框内输入,但是已失效)
  3. shop(世界商店可以买卖物品)
  4. zc(主城可以为你提供帮助)
  5. zy(资源区，可以在此获取资源)
  6. world(生存世界,可以在此随意生存)
  7. ender(假装是末地，因为0.14没有末地)

-----第八章：组织篇-----

加入组织可以帮你度过前期困难，而创建组织可以帮助其他玩家度过前期困难。

  1. 8.1---加入组织篇

    2. 组织可以帮助你有个美好的住所。

    1. 首先你需要了解每个组织的加入区在哪儿。
    2. 如果你的组织申请通过后，你需要更改组织设定的称号。
    3. 更改完称号后，你需要加入组织群，了解组织的区域。你可以在组织的区域建造庇护所。
  2. 8.2---关于创建组织

    3. 如果你觉得你足够有实力，可以创建组织。

    创建组织的资本需要至少有10w金币+用于给成员整改称号。十万金币可以让约10人加入你的组织，更改组织的称号。

    -----方法一：找PTP报备-----

    1. 前往浏览器顶部网址栏搜索grps.bbcnews.top，输入你要注册的组织名字，然后输入你的QQ号，最后进行人机验证。
    2. 领取报备号后，在群聊625015871中找到“PTP”，点开个人主页，点击[发消息]就可以找PTP报备了。
    3. 当你进入PTP的聊天界面时，你需要先把报备号截图保存，并发送给PTP，然后打开/qd菜单，并截图天数，PTP会给你登记。
    4. 创建完成

    -----方法二：找云服务报备-----

    1. 在群聊625015871中找到“cp01”，并点开个人主页，点击发消息。
    2. 和cp01发送你要注册的组织名字，干什么的，负责人QQ。
    3. 组织审核通过，创建完成。

    以上任何一种方法完成之后，你可以自由选择是否创建组织群。

    接下来是创建组织群的方法。

    首先点击右上角[3个杠]，然后点击创建群聊，点击从群聊中选择（必须要至少3个人才能创建群聊）cp01,sunch233这两人,并给予管理员权限。

-----第九章：违禁品篇-----

违禁品是服务器内最珍贵的物品资源之一，且无法通过正常途径获得，因此价格昂贵。

以下是关于违禁品的所有帮助

  1. 违禁品可能会被没收，但是可以放在屋内作为收藏品。 要获得违禁品，可以通过以下几种方式获得:
    1. 找到蜗牛(tx666),向他索要或向他购买
    2. 卡bug,绕后台

  2. 违禁品中包括红石中继器，红石分配器，这2种物品可以制作红石装置，由于服务器内禁止高频，所以就不可以使用。

  3. 获得违禁品后，一定要上交给cp01,你会获得一笔丰厚的报酬:)
