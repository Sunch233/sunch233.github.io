第一章：进入篇

对于某些人来说，加入服务器是个难题。请按你遇到的问题对症下药。

  1. 安装不上

    答：使用群文件中的[安卓14可用]版本。

  2. 服务器IP端口是什么?

    答: IP:mc-kkk.top | 端口:19132

  3. 显示login timeout

    答：这个的意思是登录时间过长，可以尝试[重启]你的[手机]。

  4. 显示“无法连接至互联网，请检查您的互联网再试”

    答: 你可能使用的是移动数据网络，这玩意不能锁定IP，无法连接。可以尝试一个有WiFi的场所或者是下载一个能用流量的客户端。如果你已经连接了互联网，可以断网重连。

  5. 显示“无法连接至世界”

    答: 重启[手机]可以解决一部分问题。

  6. 显示“名称不能用Steve”

    答: steve是游戏内的初始名称。你需要更改为除了默认名称外的任意名称，什么都行。更改教程：主界面>选项>名称>点击更改。

  7. 显示“您没有给手机存储权限”

    答: 手机设置>应用管理>找到游戏客户端(如 Minecraft - Pocket Edition 等)>权限管理>手机存储>打开权限。

第四章：常识篇

服务器内有很多东西你必须要知道，接下来是服务器内的一些机制。

  1. 重启时间：每90分钟会重启一次。
  2. 清理时间：约每10分钟会清理一次掉落物，清理提醒时间约是10秒前。
  3. 底部状态栏内容：
    • 位置(你所在的区域坐标)
    • 服务器群(加入可以获得最新动态)
    • 人数(人数超过50，服务器将异常卡顿)
    • 网络(查看当前延迟)
    • 金币(你的金币数量)
    • 方块ID(方块的编号)

第五章：指令篇

你已经可以正常游玩了，不过指令是这服务器的进阶玩法。接下来，这些是基本的指令和用途。（提示: 括号内的注释不用写）

  1. /sethome或/安家(安置你的家,安置一次花费300金币)
  2. /home或/回家(回到你安置的家)
  3. /w 世界名
    • 1.world(生存区)
    • 2.nether(地狱)(只能在前面输入/w 使用)
    • 3.ender(假装是末地)
    • 4.parkour(跑酷世界)(已失效)
    • 5.zc(主城)
    • 6.zy(资源区)
    • 7.pvp(玩家对战)
    • 8.shop(世界商店)
  4. /ws(打开世界玩家交易中心,不过一定要查看价格,有些玩家总是喜欢卖特别贵的东西,小心破产)
  5. /cd(箱子菜单,里面有很多生存用得到的功能。)
  6. /res tp 领地名(传送到指定领地)
  7. /zp i(用方块点击要查看的地方的方块状态,如果已经打开查询模式，再次输入该命令可以关闭查询模式)
  8. /tpa 玩家名称(传送到别人那里)
  9. /tpahere 玩家名称 (让别人传送到你这)
  10. /tell 玩家名称 内容(和别人说悄悄话)
  11. /prefix 想要改的称号(输入这串指令，可以让你的称号改成你喜欢的称号)
  12. 其余指令请输入/help 1~11来了解
  13. 告示牌指令如下:
    (第1行:名称可有可无)
    ( 第2行:plcmd )
    ( 第3行:命令 )
    ( 第4行:命令扩展 )

第六章：金币篇

金币是服务器里面的硬通货，你应该学会如何使用。

  1. 要把钱给别人,输入/支付 玩家名 数量
  2. 要获得免费的钱,输入/qd并点击黄色的羊毛
  3. 要通过劳动赚钱,可以前往shop世界的回收商店
  4. 要使用钱可以前往shop世界的出售商店
  5. 输入/ws可以购买其他玩家的商品,一定要注意价格,否则容易破产
  6. 想把自己的东西卖到玩家商店?手持你要卖的物品,输入/ws sell 价格
  7. 输入/查询 玩家名字 可以查看指定玩家的金币数量
  8. 输入/财富榜 可以查看财富的排行榜
  9. 服务器里面超过十万金币就需要扣税了。
    请务必注意此项！

以上就是所有关于金币的帮助了。祝你早日成为富豪！

第九章：圈地篇

圈地可以让你的家多一层保护，让你的家不会被破坏。

本服务器使用的领地插件是3d圈地，而不是像你游玩过的其他服务器使用的都是2d圈地。

  1. 第一步:做一把木锄头。

    2. 选择一片区域用于圈地。

  2. 第二步:点击一个方块作为第一个点。
  3. 第三步:挖掘一个方块作为第二个点。

    4. 接下来聊天框应该会提示类似"输入/res create 领地名称 来创建领地"的消息。

  4. 第四步:打开聊天框输入/res create [领地名称]，如果提示创建成功，那么你圈的这块领地就生效了，如果并没有，那你应该是没钱了。
